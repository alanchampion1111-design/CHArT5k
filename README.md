# CHAMPION-Parkrunners
This repo provides a Puppeteer connection (using Node.js) on a GCP server to access parkrun.org.uk site with personal certificates to capture data into the Google Spreadsheet, CHAMPION Parkrunners.

So, to gain access. I need to share this Github account.

A local copy is held in C:\dev\CHAMPION-Parkrunners (on a single device) and not on a shared Google drive to avoid race conflicts.
